# How to clone project

### 1. Clone repository

```
git clone https://github.com/mannoeu/memory-game.git
```

### 2. Change directory

```
cd memory-game
```

### 3. To install dependencies

```
yarn install
```

or, for npm users:

```
npm install
```

### 4. To run development server

```
yarn start
```

or, for npm users:

```
npm start
```

### 5. To create static files

```
yarn build
```

or, for npm users:

```
npm run build
```
